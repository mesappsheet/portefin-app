#PROTOCOLE DE TRAVAIL ANTIGRAVITY

Ce fichier définit les règles de collaboration pour l'ensemble du projet (PC et PWA).

## 📝 PROTOCOLE SYSTÉMATIQUE (À LIRE À CHAQUE DEMANDE)

À chaque nouvelle demande de l'utilisateur, l'assistant IA doit impérativement suivre ces étapes avant toute exécution technique :

1.  **ANALYSE DES BESOINS** : Je reformule précisément ce que j'ai compris de votre demande pour m'assurer qu'il n'y a aucune ambiguïté.
2.  **PLAN D'ACTION DÉTAILLÉ** : J'explique ma stratégie technique :
    *   Quels fichiers seront impactés (`MAQUETTE_COMPLETE.html`, `maquette-app/index.html`, ou les deux).
    *   Quelle logique ou structure sera implémentée.
    *   Quelles sont les conséquences éventuelles sur le reste du projet.
3.  **DEMANDE DE VALIDATION** : Je vous pose explicitement la question : "Est-ce que ce plan vous convient ? Puis-je procéder ?"
4.  **ATTENTE DU FEU VERT** : Je ne commence aucune modification de code tant que vous n'avez pas validé mon plan d'action.

---
Ce protocole est valable pour :
- **Application PC** : `MAQUETTE_COMPLETE.html`
- **Application PWA** : `maquette-app/index.html`
